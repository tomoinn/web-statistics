Circular Plot

This page can be used to produce a circular plot, with variable cycle length and binning. The plot generates a frequency histogram of your data, with the 0 point on the diagram at the North axis (i.e. where 12:00 would be on a regular wall clock).

Data, and configuration properties where applicable, are expressed in terms of hours and specified as floating point quantities, so, for example, 4:30pm would be entered as 16.5. Any data which do not match this format will be silently discarded, you have been warned!

The plot, as displayed, can be downloaded by right-clicking on it and selecting the Save image as... option (this may only work on Chrome, I've not tried any other browsers).
